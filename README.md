## Mobx Learning

本项目用于初学 [Mobx](https://mobx.js.org/) 状态管理工具，只在 `deveopment` 模式下查看实验效果。

### 运行

```bash
cd mobx-learning
npm install
npm start
```

### 基础知识

- Webpack 配置
- Babel Plugins
- decorator 语法

### Mobx 简单语法

- observable
- computed
- autorun
- when
- reaction
- action
